静态资源服务器

基于 Node.js 原生 http 模块构建

TypeScript 零依赖运行 DDD 架构

✨ 核心特性

📁

静态文件托管

自动映射 URL 路径到本地文件系统，支持目录浏览

🔍

MIME 识别

覆盖 40+ 文件类型，自动设置正确的 Content-Type

🛡️

安全防护

路径遍历攻击检测与拦截，保障服务器安全

⚡

ETag 缓存

支持 ETag / Last-Modified 缓存协商，减少不必要的传输

📦

Gzip 压缩

文本资源自动启用 Gzip 压缩，显著减小传输体积

📡

Range 请求

支持断点续传和大文件分片传输（206 Partial Content）
